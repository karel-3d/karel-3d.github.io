 Sheep Changelog (daily updated) Sun Oct 27, 2013 / 05:57 PM

Reply message Sheep Changelog (daily updated)
24/10

- remake of refund system (now % goes allways to buyer)
- fixed shopping cart bug


25-26/10

- fixed the bug when marked multiple orders to approved was only changing for first order
- all items in shop have now two prices, e.g., EUR/BTC shown
- fixed that item which was sold out was stil showing on vendor page
- fixed bug that edited feedback didnt update on seller/buyer profile
- on mainpage now only show sellers with 95% and more rating
- new sellers doesnt have 100% rating but rather N/A (blue colored) untill they get 25 positive ratings


27/10

- [seller] disabled item is now not only hidden link but really disabled (plus orders in cart of any buyer which are not paid will be removed)
- user on forum can now edit own post
- [seller] removed autoverification process for now (50 positive feedback) [todo add fake feedback checks and some more]
- forum have now 25 topics in each category and 25 posts in each topic per page (up from 15 previously)
- number of active forum pages (at bottom) is now marked

28-29/10

- market/web/forum code optimalization (preparing for big maintenance next week)
- configuring few new servers for our infrastructure (not connected yet)

31/10

- newbie section on forums (some guides and tutorials will follow soon)
- digital item bug fixes


Due to the extreme load that Sheep Marketplace is experiencing, we will have to change webserver and database technology. It needs some code updates, so our programmers will only be focussing on urgent bug fixes for now.

You can expect an extended maintenance period, probably during next weekend (9-10/11). It may take few hours to rebuild our whole infrastructure and add new servers. You will be informed as soon our programmers finish working on the code.
