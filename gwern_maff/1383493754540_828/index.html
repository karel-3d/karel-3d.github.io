Nette Framework

Populární nástroj pro vytváření webových aplikací v PHP 5. Při programování vám vychází vstříc a nepřidělává vrásky. Eliminuje bezpečnostní rizika, podporuje AJAX a znovupoužitelnost.

Verze 2.0 je venku

Po letech vývoje, po alfa a beta verzích a třech release kandidátech, s hrdostí oznamujeme, že vyšel Nette Framework 2.0 final. Stahujte!

Vyzkoušejte Nette!

Framework vám ulehčí práci, budete méně psát, mít přehlednější kód a radost z práce. Začněte používat Nette Framework.

Poslední sobota

Přijďte na Poslední sobotu, pravidelné neformální setkání přátel Nette Frameworku, střídavě pořádané v Praze, Brně a nebo jinde.

Dokonalé zabezpečení

Nette používá revoluční technologii, která eliminuje výskyt bezpečnostních děr a jejich zneužití, jako je např. XSS, CSRF, session hijacking, session fixation atd. Chcete vědět více?

Ladicí nástroje

Disponuje bezkonkurenčními ladicími nástroji, které vám pomohou zavčasu odhalit všechny chyby aplikace.

Exceluje ve výkonu

Dle nezávislého testu je Nette Framework jedním z nejrychlejších frameworků vůbec.

Nejaktivnější komunita v ČR

Nebudete v tom sám! Komunita kolem frameworku radí, vytváří doplňky a rozšíření, sdílí své zkušenosti.

Strmá křivka učení

Naučíte se tvořit webové aplikace za velmi krátký čas.

Dobré návyky

Nette Framework vás vede k čistému návrhu aplikace s důrazem na budoucí rozšiřitelnost.

Moderní framework

AJAX / AJAJ, Dependency Injection, SEO, DRY, KISS, MVC, Web 2.0, cool URL – propracovaná podpora všech moderních technologií a koncepcí.

Vyzrálý objektový návrh

Promyšlený a čistý objektový návrh využívající nových vlastností PHP 5, komponent a událostmi řízeného modelování.

Pluginy a rozšíření

Neustále se rozrůstající nabídka doplňků pro snadné použití ve vašich aplikacích. Oceňte výhody znovupoužitelného kódu.

Open-source licence

Licence BSD patří k těm nejvolnějším. Framework můžete zdarma používat i v komerčních projektech.

Práce v týmu

Výborně rozděluje práci mezi více programátorů a HTML kodérů.

Dává vám volnost

Nette Framework vás nijak neomezuje. Dokonce jej lze výhodně kombinovat například se Zend Frameworkem.
